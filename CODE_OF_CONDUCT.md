# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and professional environment for
everyone contributing to the VBW project.

## Our Standards

**Expected behavior:**
- Professional, constructive communication
- Respectful disagreement focused on technical merit
- Accepting responsibility for mistakes and learning from them
- Prioritizing the security and correctness of the project

**Unacceptable behavior:**
- Harassment, intimidation, or discrimination of any kind
- Publishing others' private information without consent
- Deliberately introducing security vulnerabilities or backdoors
- Sustained disruption of project discussions or processes

## Enforcement

Instances of unacceptable behavior may be reported to: **conduct@scqcs.dev**

Project maintainers will review all reports and take appropriate action,
which may include warning, temporary ban, or permanent removal from the project.

## Scope

This Code of Conduct applies to all project spaces including GitHub issues,
pull requests, discussions, and any public forum where an individual
represents the project.
